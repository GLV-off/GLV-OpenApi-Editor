# Сценарии

## Сценарий - Первый запуск

Предусловия Программа только установлена\скопирована на носитель пользователя

Когда программа запущена в первый раз
То Программа загружает конфигурацию по умолчанию

## Сценарий - Закрытие программы

Когда пользователь закрывает программу
То В файл конфигурации cfg.ini записываются опции принятые во время работы программы
